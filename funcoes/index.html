Para declaram uma função precisamos usar a palavra reservada function e o nome da função o que ela faz, fica melhor.

Funções podem ser void quando não retornam nada, apeanas processam uma logíca

Compreendendo a Essência das Funções com Parâmetros:

Em sua essência, funções com parâmetros em JavaScript são blocos de código reutilizáveis que aceitam entradas definidas, chamadas de parâmetros, e as processam para gerar um resultado útil. Esses parâmetros fornecem à função a flexibilidade de se adaptar a diferentes cenários e realizar tarefas específicas.
