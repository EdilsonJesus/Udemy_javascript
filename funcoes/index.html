Funções Principio

function calcularAreaTerreno(largura,comprimento){
var area = largura * comprimento
return area
}

Para declaram uma função precisamos usar a palavra reservada function e o nome da função o que ela faz, fica melhor.
Alem disso, toda função tem que um parametros, que nada mais é que entrada de dados para as funções.
No Escopo da função, podemos criar variáveis, laços de repetição.

Funções podem ser void quando não retornam nada, apeanas processam uma logíca

Ou retornarem uma expressão, como vemos acima

Em resumo, quando uma função é chamada alem de processar alguma lógica, ela poderar dar um Retorno de valor pre definido
